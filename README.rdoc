== README

Need to have elasticsearch installed
Redis

To deploy:

Run

gem install elasticsearch-model elasticsearch-rails

set AWS S3 bucket keys

rake secret

bundle install

rake db:migrate
rake recipes:all
rake recipe_tags:all
rake members:all
